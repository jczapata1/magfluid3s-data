<h1 align="center">MagFluid3S Data</h1>

<p align="justify">
This repository stores data and results related to magnetic fluids stochastic simulations. The data has been generated with the <a href="https://github.com/jczapata1/magfluid3s" style="color:#0366d6;">MagFluid3S</a> framework. Thank you for your visit ✌️.
</p>
